# TaskFlow - Collaborative Project Management API (Full)
This repository contains a ready-to-run Node.js + Express + MySQL backend
for TaskFlow. It includes migrations, sample data, controllers, routes,
basic models, middleware, simple tests and websocket stubs.

## Setup
1. Ensure MySQL is running and create a database (name in .env).
2. Run the SQL in migrations/create_tables.sql then migrations/sample_data.sql
3. Install dependencies: `npm install`
4. Start server: `npm run dev`
5. Run tests: `npm test`
